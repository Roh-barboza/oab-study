# OAB Study Premium

Sistema de estudos gamificado para a OAB — desenvolvido para estudar por matéria, revisar com flashcards, acompanhar acertos e erros em tempo real e manter uma rotina consistente com calendário semanal.

## Funcionalidades

- Dashboard com XP, streak, acertos e erros
- Cards por matéria com progresso individual
- Questões com feedback imediato e marcação de revisão
- Flashcards por disciplina
- Modos de estudo: simulado, revisão de erros, missão diária
- Calendário semanal de estudo (Segunda a Domingo)
- Timer Pomodoro integrado
- Mapa de calor semanal
- Conquistas e gamificação
- Modo escuro/claro

## Como publicar na Vercel

1. Acesse [vercel.com](https://vercel.com) e entre com sua conta GitHub
2. Clique em **Add New → Project**
3. Selecione o repositório `oab-study`
4. Clique em **Deploy** — pronto!
